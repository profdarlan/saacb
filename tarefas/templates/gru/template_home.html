ℹ️ Informações Rápidas

📌 O que é GRU?

Guia de Recolhimento da União é um instrumento usado para recolher receitas devidas à União, como multas, taxas e contribuições.

🔗 Integração

Este sistema consulta a API SISGRU do Governo Federal para fornecer dados em tempo real sobre suas GRUs.

⏱️ Disponibilidade

A API está disponível segunda a sexta, das 08:00 às 22:00 (horário de Brasília), exceto feriados.

📖 Como Usar

  1. Acesse "Consultar GRU" para fazer sua primeira consulta
  2. Forneça suas credenciais do Conecta.Gov.BR (usuário e senha)
  3. Digite o número da GRU (32 dígitos)
  4. Clique em "Consultar" e aguarde os dados
  5. Baixe o PDF se desejar uma cópia visual
  6. Acompanhe no histórico todas suas consultas anteriores

❓ Perguntas Frequentes

Preciso ter credenciais do Conecta.Gov.BR?

Sim, você precisará ter um usuário e senha registrados no Conecta.Gov.BR para acessar a API.

Como obtenho meu número de GRU?

O número da GRU é fornecido pelo órgão responsável pela arrecadação ou aparece em documentos oficiais relacionados à receita.

Os PDFs gerados são válidos?

Sim! Os PDFs são gerados com base nos dados da API SISGRU oficial do Governo Federal.

Meus dados ficam salvos?

Se você ativar o módulo de histórico, as consultas são salvas no banco de dados. Caso contrário, são consultadas apenas em sessão.
